EVTouch 4point

EVTouch's 4 point calibrator for GNU/Linux

Calibration tool do not require setting of special options in xorg.conf or starting Xorg in special mode.

Based on http://linux.chapter7.ch/touchkit/.

Supported hardware:

  • GeneralTouch
  • ATouch
  • Digital Vision

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/gepo/evtouch-4point

Installation

$ mkdir build && cd build
$ cmake ..
$ make
get the source code on GitHub : gepo/evtouch-4point
